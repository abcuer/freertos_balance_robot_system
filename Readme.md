# 全能平衡车PLUS: 遥控 | 跟随 | 安全保护 | UI设计
 本项目是一款基于 **STM32** 与 **FreeRTOS实时操作系统**开发的多功能双轮平衡车。通过高效的**任务调度/信号量通信与逻辑设计**，实现了稳定的**平衡控制**、高精度的**环境感知**以及直观的 **u8g2 UI** 交互。
